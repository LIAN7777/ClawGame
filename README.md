# ClawGame

休闲养成小游戏 - 卡通风格

## 项目结构

```
ClawGame/
├── src/                # 游戏源代码
│   ├── main.py        # 游戏入口
│   ├── game/          # 游戏核心模块
│   ├── ui/            # 用户界面
│   └── utils/         # 工具函数
├── tests/             # 测试文件
├── assets/            # 游戏资源
│   ├── images/        # 图片资源
│   ├── sounds/        # 音效资源
│   └── fonts/         # 字体文件
├── docs/              # 项目文档
├── .github/           # GitHub 配置
│   └── workflows/     # CI/CD 工作流
└── requirements.txt   # Python 依赖
```

## 开发环境

- Python 3.12+
- Pygame

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 运行游戏
python src/main.py
```
