# ClawGame 项目文档

## 项目概述

**ClawGame** 是一个休闲养成类小游戏，采用卡通风格，使用 Pygame 开发。

- **GitHub**: https://github.com/LIAN7777/ClawGame
- **本地路径**: `/root/.openclaw/workspace-clawgame`
- **技术栈**: Python 3.12 + Pygame

## 运行方式

### Windows
```powershell
git clone https://github.com/LIAN7777/ClawGame.git
cd ClawGame
pip install -r requirements.txt
python src/main.py
```

### 操作
| 按键 | 功能 |
|------|------|
| WASD/方向键 | 移动 |
| Shift | 奔跑 |
| 空格 | 跳跃 |
| E | 与NPC交互 |
| ESC | 暂停 |
| Q | 退出 |
