# ClawGame 每日记忆

## 2026-03-13

### 🐛 紧急修复：游戏启动画面异常

**用户反馈：**
游戏完全不可用，界面是大片色块堆叠，没有人物、NPC，没有任何可操作空间，连退出键都没反应。

**问题分析：**
1. **模块导入路径错误**：`main.py` 使用 `from config import config`，但运行 `python -m src.main` 时 Python 路径不正确
2. **相机尺寸错误**：`Game` 类使用窗口尺寸（800x600）初始化相机，但实际渲染到内部表面（400x300）
3. **相机初始位置问题**：相机初始位置是 (0,0)，玩家出生在 (240,336)，且使用平滑跟随，第一帧玩家不在视口内

**修复内容：**
- `src/main.py`：添加 Python 路径设置，支持 `python -m src.main` 运行
- `src/game/game.py`：
  - 使用 `config.internal_width/height` 替代 `screen.get_width/height`
  - 相机跟随时立即调用 `update()` 定位
  - 添加 `from config import config` 导入
- `tests/test_game.py`：更新测试以匹配新的 Game 类行为

**验证结果：**
- 66 个测试全部通过
- 相机位置：(52.0, 84.0) - 合理
- 相机尺寸：400x300 - 正确匹配内部渲染尺寸
- 玩家在视口内：True
- 所有 3 个 NPC 在视口内：True
