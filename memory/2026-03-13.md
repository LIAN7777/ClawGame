# ClawGame 每日记忆

## 2026-03-13

### 🐛 紧急修复：游戏启动画面异常

**用户反馈：**
游戏完全不可用，界面是大片色块堆叠，没有人物、NPC，没有任何可操作空间，连退出键都没反应。

**问题分析：**
1. **模块导入路径错误**：`main.py` 使用 `from config import config`，但运行 `python -m src.main` 时 Python 路径不正确
2. **相机尺寸错误**：`Game` 类使用窗口尺寸（800x600）初始化相机，但实际渲染到内部表面（400x300）
3. **相机初始位置问题**：相机初始位置是 (0,0)，玩家出生在 (240,336)，且使用平滑跟随，第一帧玩家不在视口内

**修复内容：**
- `src/main.py`：添加 Python 路径设置，支持 `python -m src.main` 运行
- `src/game/game.py`：
  - 使用 `config.internal_width/height` 替代 `screen.get_width/height`
  - 相机跟随时立即调用 `update()` 定位
  - 添加 `from config import config` 导入
- `tests/test_game.py`：更新测试以匹配新的 Game 类行为

**验证结果：**
- 66 个测试全部通过
- 相机位置：(52.0, 84.0) - 合理
- 相机尺寸：400x300 - 正确匹配内部渲染尺寸
- 玩家在视口内：True
- 所有 3 个 NPC 在视口内：True

### 🐛 修复移动键无响应和FPS显示问题

**用户反馈：**
可以看到场景和人物了，四个不同颜色但风格完全相同的角色。按移动键没有反应，FPS显示一直为0。

**问题分析：**
1. **玩家出生点碰撞问题**：出生点在门前一格(7,10)，碰撞盒覆盖了墙壁格子(8,11)，导致向下和向右移动被阻止
2. **FPS计算错误**：`Game` 类有独立的 `_clock` 但从未调用 `tick()`，`get_time()` 始终返回0
3. **主角辨识度低**：主角和NPC使用相似的粉色调，难以区分

**修复内容：**
1. `src/game/scene.py`：
   - 出生点从门前一格改为门前两格，避免与墙壁碰撞
   
2. `src/game/game.py`：
   - 移除 `_clock`，改用 `_last_dt` 存储外部传入的 delta time
   - `update()` 方法接收 `dt` 参数
   - `_render_debug_info()` 使用 `1/dt` 计算FPS
   - 调试文字改为白色

3. `src/main.py`：
   - 在游戏循环中计算 `dt` 并传给 `game.update()`

4. `src/game/entity/player.py`：
   - 主角配色改为更鲜艳的珊瑚粉
   - 添加金色小皇冠作为主角标识

**验证结果：**
- 玩家位置：(240, 304) - 不与墙壁碰撞
- 四个方向移动测试：全部返回 True
- FPS计算：dt=0.016 → FPS=62 ✅
- 66个测试全部通过

### ✨ 优化游戏体验：修复黑线闪烁、交互卡顿、移除调试信息

**用户反馈：**
1. 屏幕上时不时出现横屏或竖屏的细黑线，过一会又会消失
2. 主角与NPC按E互动时完全没有反应且游戏会卡住一会
3. 屏幕左上角FPS和主角位置字幕显示太占位置且不美观

**问题分析：**
1. **细黑线闪烁**：tilemap渲染时使用浮点数坐标，导致tile之间出现缝隙
2. **交互卡住**：没有交互冷却时间，快速重复触发导致问题
3. **调试信息**：`_render_debug_info` 一直被调用

**修复内容：**
1. `src/game/tilemap.py`：
   - 相机偏移使用 `int()` 转换为整数，避免浮点数缝隙

2. `src/game/interaction.py`：
   - 添加 `_interact_cooldown` 冷却时间（0.5秒）
   - 添加 `update_cooldown()` 方法
   - `try_interact()` 检查冷却时间

3. `src/game/entity/npc.py`：
   - 添加 `DIALOGS` 字典，不同颜色NPC有不同对话
   - `interact()` 方法防止重复触发
   - 气泡显示时间从2秒延长到3秒

4. `src/game/game.py`：
   - 移除 `_render_debug_info()` 调用
   - 调用 `interaction_system.update_cooldown(dt)`

**NPC对话内容：**
- 绿色: "你好呀~", "今天天气真好！", "需要帮忙吗？"
- 蓝色: "嘿~", "有什么新鲜事吗？", "一起来玩吧！"
- 黄色: "你好你好~", "见到你真开心！", "有什么想聊的吗？"

**验证结果：**
- 66个测试全部通过 ✅
- 交互系统正常工作 ✅
- NPC对话正常显示 ✅
